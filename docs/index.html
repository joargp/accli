accli

Apple Calendar
from the command line

A fast, elegant CLI for managing your macOS calendar. Built with JXA and EventKit.

$ npm i -g @joargp/accli
macOS only · MIT License

Features

Everything you need, nothing you don't

Full Calendar Control

List, create, update, and delete events. Query by date range, search text, or event ID.

Agent-Ready

Built for coding agents: structured JSON output, distinct exit codes, machine-readable errors, and persistent IDs.

Free/Busy Queries

Check availability across multiple calendars at once. Perfect for scheduling tools.

Native & Secure

Uses macOS EventKit via JXA. Your data never leaves your machine.

Configurable Defaults

Set a default calendar once, then run commands without specifying it every time.

Fast & Lightweight

Zero runtime dependencies. Instant startup. Does one thing and does it well.

Usage

Simple, intuitive commands

$ accli setup # Grant calendar permissions
$ accli calendars # List all calendars
$ accli config set-default # Set default calendar
$ accli events --from 2025-01-01 # List upcoming events
$ accli create --summary "Team sync" --start 2025-01-15T10:00 --end 2025-01-15T11:00
$ accli freebusy --from 2025-01-15 --to 2025-01-16 --json
